# Copywriting & Content Design

The words are half the design system. This is the human companion to the
`writing-gsc-copy` skill — same rules, fuller context. If you write anything
GSC-facing, this is the guide.

**Skill:** `.claude/skills/writing-gsc-copy/` (AI agents read this automatically)

## The core idea

**Clarity beats cleverness. Write the CTA first.** GSC sells football-club
investment to sophisticated people. The prose is investment-grade, confident,
and plain — numbers do the bragging.

## Voice vs tone

- **Voice is constant** — *the disciplined operator who has already done the work.* Serious (with dry restraint), lightly formal, respectful, matter-of-fact.
- **Tone flexes** along one axis — *warmer ⟷ more sober* — by context.

| Context | Tone | Example |
|---|---|---|
| Hero / thesis | Bold, declarative | "A crown no one else wears." |
| Body / analysis | Sober, evidenced | "Priced at 1.4× revenue — below the league median." |
| CTA | Direct, low-pressure | "Request the data room." |
| Error | Calm, plain, no blame | "That email didn't go through. Try again." |
| Success | Understated | "You're on the list. We'll be in touch." |
| Legal / risk | Precise, complete | "Football investment involves risk, including loss of capital." |

## The seven copy principles

1. Numbers do the bragging.
2. One idea per sentence.
3. Say the specific thing.
4. Verbs over adjectives.
5. Headlines are claims, not topics.
6. CTAs name the action.
7. Ration the urgency.

## Lexicon (short version)

**Use:** asset, opportunity, thesis, priced below its history, diaspora, reach,
recurring revenue, media rights, discipline, track record, request the data room,
the window is open.

**Avoid:** world-class, game-changing, revolutionary, disruptive, unlock,
leverage, synergy, seamless, robust, ecosystem; very/really/incredibly;
exclamation stacks; emoji in investor materials.

**Bilingual (EN/ES):** GSC is Guatemalan/Central American. Investor materials
default to English; local/club/diaspora campaigns to Spanish — written natively,
never machine-translated. Prefer *afición, aficionados, palmarés, cantera,
derechos de transmisión, afición en el exterior.* Tagline stays *Live the
dream.* (or *Vive el sueño.* locally — never both at once).

## Mechanics

- Figures for stats (`30+`, `1.4×`, `$5M`); spell out only at sentence start.
- Multiplication sign `×`, not "x".
- Display headlines are ALL CAPS by typographic design (not shouting); body is sentence case; labels are tracked caps.
- One exclamation mark maximum, reserved for the tagline.
- "Global Sports Corporation" first use, "GSC" after — never "the GSC".

## Microcopy quick rules

- **Buttons:** verb + object ("Request the data room"), never "Submit"/"Learn more".
- **Errors:** what happened + what to do, no codes, no blame.
- **Empty states:** what goes here + first action; never a dead end.
- **Success:** confirm + set expectation; no confetti.
- **Forms:** real labels, example placeholders, minimum fields.
