# Lexicon — words to use / avoid

GSC operates in Guatemala and Central America; audiences are bilingual. Keep a
parallel EN/ES register. Default to English for investor materials, Spanish for
local/club-facing and diaspora campaigns — never machine-literal translations.

## Use these (EN)

| Prefer | Over |
|---|---|
| asset, opportunity, thesis | deal, play, bet |
| priced below its history | undervalued (vague) |
| diaspora, reach, audience | eyeballs, fanbase |
| recurring revenue, media rights | monetization, revenue streams |
| discipline, operator, track record | strategy, vision, DNA |
| request the data room | learn more, submit, get started |
| the window is open | act now, don't miss out |

## Avoid these (EN)

- **Hype:** world-class, best-in-class, game-changing, revolutionary, disruptive, unparalleled, unlock, supercharge, next-level.
- **Filler:** leverage, synergy, seamless, robust, holistic, ecosystem (unless literal), solutions.
- **Weak intensifiers:** very, really, incredibly, hugely, truly.
- **Punctuation tells:** exclamation stacks (`!!!`), ALL-CAPS shouting in body copy, emoji in investor materials.

## Spanish register (ES)

- Voice carries over: sober, specific, respetuosa. Usa "usted" en materiales de inversión; "vos/tú" solo en campañas locales según región.
- Tagline stays **"Live the dream."** in English as a brand lockup; if translated for local warmth: *"Vive el sueño."* — never both at once.
- Prefer: activo, oportunidad, afición, aficionados, palmarés (titles/honours), cantera (academy), derechos de transmisión (media rights), afición en el exterior (diaspora).
- Avoid anglicisms where a clean Spanish word exists (di "afición", no "fanbase").

## Mechanics

- **Numbers:** use figures for stats ("30+ titles", "1.4×", "$5M"). Spell out only at sentence start.
- **Multipliers:** `1.4×` with the multiplication sign, not "1.4x".
- **Capitalization:** headlines in the display font are ALL CAPS by design (that's typographic, not shouting). Body is sentence case. Labels are Title/UPPERCASE tracked.
- **Punctuation:** one exclamation mark maximum, and only in the tagline. Use the em dash — like this — for asides.
- **Brand name:** "Global Sports Corporation" on first use, "GSC" after. Never "G.S.C." or "the GSC".
