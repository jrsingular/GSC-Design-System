# Writing GSC Copy

GSC sells football-club investment opportunities to sophisticated investors.
The writing is **investment-grade, confident, and plain**. Numbers do the
bragging; the prose stays calm. Tagline: *Live the dream.*

## The one rule

**Clarity beats cleverness. Write the CTA first.** Know the single action a
piece of copy must drive, then make every sentence lead there. If a reader has
to read a line twice, rewrite it.

## Voice (constant) vs tone (flexes)

**Voice** — who GSC always is: a disciplined operator who has done the work.
Certain, unhurried, specific. Never hypey, never salesy, never jargon-drunk.

**Tone** — flexes by context along one axis: *warmer* ⟷ *more sober*.

| Context | Tone | Example |
|---|---|---|
| Hero / thesis | Bold, declarative | "A crown no one else wears." |
| Body / analysis | Sober, evidenced | "Priced at 1.4× revenue — below the league median." |
| CTA | Direct, low-pressure | "Request the data room." |
| Error / problem | Calm, plain, no blame | "That email didn't go through. Check the address and try again." |
| Success | Brief, understated | "You're on the list. We'll be in touch." |
| Legal / risk | Precise, unembellished | "Football investment involves risk, including loss of capital." |

## Copy principles

1. **Numbers do the bragging.** Prefer "30+ titles" to "incredibly successful."
2. **One idea per sentence.** Short declaratives. Cut qualifiers.
3. **Say the specific thing.** "Three million diaspora fans in the U.S.", not "a huge global audience."
4. **Verbs over adjectives.** "Renegotiate the media rights," not "exciting media opportunities."
5. **Headlines are claims, not topics.** "The window is open" beats "Market timing."
6. **CTAs name the action.** "Request the data room," never "Submit" / "Learn more."
7. **Ration the red words.** Urgency ("now", "open window") is earned, used once.

## Workflow

Copy this checklist when drafting:

```
- [ ] Identify the single CTA / desired action
- [ ] Draft the headline as a claim (not a label)
- [ ] Write body: one idea per sentence, numbers where possible
- [ ] Pass the lexicon check (words to use / avoid)
- [ ] Set tone to the context (warmer ⟷ sober table above)
- [ ] Read aloud — cut anything you'd never say to one smart person
- [ ] Run the compliance checklist below
```

## Compliance checklist (validate every draft)

- [ ] There is exactly one clear primary action.
- [ ] The headline is a claim, not a topic label.
- [ ] Every stat is specific and, where possible, benchmarked.
- [ ] No banned words (see lexicon): "leverage synergies", "world-class", "revolutionary", "!" stacks.
- [ ] CTA names the action ("Request the data room").
- [ ] Tone matches context (calm on errors, understated on success).
- [ ] Reads naturally aloud to one person.
- [ ] Risk/legal lines are precise and present where required.

## Evaluations

1. *"Write a hero headline + subhead for a new club acquisition in Costa Rica."* → Claim headline, one specific number, a low-pressure CTA.
2. *"Rewrite this error: 'Invalid input error 400.'"* → Calm, plain, actionable, no blame or codes.
3. *"Draft the CTA-band copy for the data-room request."* → One action, urgency used once, risk line present.
